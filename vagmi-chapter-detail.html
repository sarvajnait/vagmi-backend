What is Percentage?

Percentage means "per hundred." It is a way of expressing a number as a fraction of 100. The symbol % is used to denote percentage.

For example, 45% means 45 out of 100, or 45/100 = 0.45

📐
Key Formula
Percentage = (Value / Total) × 100
Value = (Percentage × Total) / 100

Percentage Change

When a value increases or decreases, we calculate the percentage change to understand the magnitude of the change relative to the original value.

📐
Key Formula
% Change = [(New − Old) / Old] × 100
Increase → positive, Decrease → negative
⚡
Shortcut
If a number increases by x% then decreases by x%, the net effect is always a decrease of (x²/100)%

Successive Percentage

When two percentage changes happen one after another on the same base, we use the successive percentage formula instead of adding them directly.

📐
Key Formula
Net effect = a + b + (ab/100)
Where a and b are successive % changes
⚠️
Common Mistake
Students often add successive percentages directly (20% + 30% = 50%). Always use the formula — the correct answer is 56%.
